getpgrp() - zwraca ID grupy procesowej procesu wywołującego. 
setpgrp() - jeśli proces wywołujący nie jest liderem sesji, setpgrp() ustawia ID grupy procesowej procesu wywołującego na ID procesu wywołującego. Jeśli proces wywołujący funkcję jest liderem, wywołanie funkcji nie przynosi żadnych efektów. 

geteuid() - zwraca efektywny ID użytkownika wywołującego proces.
getegid() - zwraca efektywny ID grupy wywołującej proces.
